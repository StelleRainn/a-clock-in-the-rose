# A Clock Inside the Rose (ACIR) - 里程碑报告

---

## 📅 Milestone 2.0: 游戏化与个性化 (Current)
**版本号**: v0.3.0-beta
**完成日期**: 2026-03-07
**主要成就**: 引入“游戏化”机制（等级、成就、连胜），增强用户粘性；完善用户个性化设置与社交分享功能，打造更具温度的产品体验。

### 🚀 新增核心特性 (Key Features)

#### 1. 游戏化系统 (Gamification System)
- [x] **等级系统 (Leveling)**: 
    - 专注 1 分钟 = 1 XP，自动升级，难度曲线平滑。
    - 顶栏动态展示等级徽章与进度条（Lv.5 1024/2048），颜色随等级变化。
- [x] **成就系统 (Achievements)**: 
    - 数据库预设成就（如 "Focus Master"），后端自动根据用户数据判断解锁。
    - 统计页面新增“成就墙”，未解锁成就以神秘灰色展示。
- [x] **连胜纪录 (Streaks)**: 
    - 自动追踪连续专注天数。
    - Dashboard 新增动态火焰卡片 (🔥)，日历视图中点亮连胜日期。

#### 2. 个性化中心 (Personalization)
- [x] **用户资料 (Profile)**: 
    - 新增 `/profile` 设置页，支持修改头像 URL、昵称、性别、简介、个人网站。
    - 全局 UI（顶栏、欢迎语）优先展示用户昵称与自定义头像。

#### 3. 社交分享 (Social Sharing)
- [x] **专注战报 (Focus Card)**: 
    - 一键生成精美图片，包含用户头像、等级、专注总时长、连胜天数及最新成就。
    - 利用 `html2canvas` 实现前端截图，保护隐私，即生成即保存。

## 📅 Milestone 1.5: 体验升级与爆款功能 (Previous)
**版本号**: v0.2.0-beta
**完成日期**: 2026-03-06
**主要成就**: 在核心功能基础上，实现了四大增强体验的“爆款”功能，显著提升了系统的互动性和沉浸感。

### 🚀 新增核心特性 (Key Features)

#### 1. 交互式任务看板 (Interactive Kanban)
- [x] **拖拽管理**: 引入 `vuedraggable`，实现任务卡片在 `To Do` / `In Progress` / `Done` 状态列之间的自由拖拽。
- [x] **实时同步**: 拖拽即触发状态更新 API，配合乐观更新 (Optimistic UI) 策略，操作丝般顺滑。
- [x] **视觉优化**: 独立的看板视图，支持通过卡片颜色快速识别优先级。

#### 2. 沉浸式专注模式 (Immersive Focus Mode)
- [x] **全屏体验**: 一键切换至全屏深色模式，隐藏所有非必要 UI 元素，打造纯净专注空间。
- [x] **环境白噪音**: 内置 **Rain** (雨声) 和 **Cafe** (咖啡馆) 两种白噪音，支持即点即播。
- [x] **极简 UI**: 放大倒计时显示，加宽进度条，提供沉浸式的视觉反馈。

#### 3. 专注与任务绑定 (Task Binding)
- [x] **任务选择**: 启动番茄钟前，支持从下拉列表中选择当前要攻克的具体任务。
- [x] **状态反馈**: 在沉浸模式下醒目展示 `"Focusing on: [Task Name]"`，强化目标感。
- [x] **数据关联**: 专注记录 (`PomodoroRecord`) 中新增 `taskId` 字段，为后续的精细化统计打下基础。

#### 4. 仪表盘日历视图 (Dashboard Calendar)
- [x] **可视化日历**: 在仪表盘首页集成交互式日历组件。
- [x] **多维展示**:
    - 🔴 **红点**: 标记当天有截止任务。
    - 🟢 **绿标**: 标记当天已积累的专注时长。
- [x] **信息聚合**: 悬停查看详情，直观对比每日的“计划”与“执行”情况。

## 📅 Milestone 1: 核心业务闭环 (Previous)
**版本号**: v0.1.0-alpha
**完成日期**: 2026-03-05
**主要成就**: 完成核心业务闭环（用户认证 -> 任务管理 -> 番茄钟专注 -> 数据统计），前后端基础架构搭建完毕。

### 🚀 核心功能交付 (Delivered Features)

### 1. 用户认证模块 (Authentication)
- [x] **后端**:
    - 基于 MySQL 的用户表设计 (`users`)。
    - 实现了 `/api/auth/register` (注册) 和 `/api/auth/login` (登录) 接口。
- [x] **前端**:
    - 登录/注册页面 (`LoginView.vue`)，支持表单验证。
    - 基于 Pinia 的 `UserStore`，管理用户登录状态和 Token（目前模拟）。
    - 解决了路由代理 404 问题，确保前后端通信顺畅。

### 2. 任务管理模块 (Task Management)
- [x] **后端**:
    - 任务实体 (`Task`) 与数据库表 (`tasks`) 设计。
    - 完整的 CRUD 接口 (`/api/tasks`)，支持按用户 ID 查询。
- [x] **前端**:
    - 任务列表视图 (`TaskListView.vue`)，支持展示任务状态、优先级（颜色标签）。
    - 任务创建/编辑弹窗，支持设置标题、描述、优先级、状态和截止时间。
    - 删除了未登录创建任务导致的空指针异常，增强了健壮性。

### 3. 番茄钟专注模块 (Pomodoro Timer)
- [x] **后端**:
    - 专注记录实体 (`PomodoroRecord`) 与数据库表 (`pomodoro_records`) 设计。
    - 记录保存接口 (`POST /api/pomodoro`) 与今日统计接口 (`GET /api/pomodoro/today-count`)。
- [x] **前端**:
    - **倒计时逻辑**: 基于 Pinia 的 `PomodoroStore`，支持开始/暂停/重置。
    - **状态持久化**: 利用 `localStorage` 实现防刷新机制，页面刷新后倒计时自动恢复。
    - **自定义时长**: 引入滑块组件 (`el-slider`)，允许用户在 1-60 分钟内自由调整专注时长。
    - **数据同步**: 倒计时结束后自动调用后端接口保存记录，并实时更新“今日完成数”。

### 4. 数据统计模块 (Statistics)
- [x] **后端**:
    - 聚合查询接口：每日专注时长 (`/api/stats/focus-daily`) 与 任务状态分布 (`/api/stats/task-status`)。
- [x] **前端**:
    - 集成 **ECharts** 图表库。
    - **专注趋势图**: 柱状图展示过去 7 天的每日专注时长。
    - **任务分布图**: 饼图展示 To Do / In Progress / Done 的任务占比。

### 5. 仪表盘 (Dashboard)
- [x] **首页聚合**:
    - 展示“我的任务”列表（最新 5 条）。
    - 展示“今日专注”统计数据。
    - 顶部栏显示当前登录用户名，支持注销。

## 🛠 技术架构 (Technical Architecture)

- **前端**: Vue 3 + Vite + Element Plus + Pinia + Axios + ECharts + Sass + VueDraggable + html2canvas
- **后端**: Spring Boot 3.2.3 + MyBatis 3.0.3 + MySQL 8.0
- **数据库**: 关系型数据库，包含 `users`, `tasks`, `pomodoro_records`, `user_stats`, `achievements`, `user_achievements` 六张核心表。
