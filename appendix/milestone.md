# A Clock Inside the Rose (ACIR) - 里程碑报告

---

## 📅 Milestone 3.0: 深度与专业化 (Current)
**版本号**: v0.4.0-beta
**完成日期**: 2026-03-12
**主要成就**: 从“好用”到“专业”的跨越。实现了数据开放（导出）、任务深度管理（标签、子任务、筛选）、可视化分析（热力图、Rose Chart）以及流畅的交互体验（拖拽排序、Zen Note），显著提升了系统的生产力属性。

### 🚀 新增核心特性 (Key Features)

#### 1. 任务标签系统 (Task Tags)
- [x] **多彩标签**: 
    - 创建任务时可直接输入并创建新标签（自动分配颜色）。
    - 列表和看板视图中均展示标签徽章，视觉识别度高。
- [x] **多维管理**: 
    - 支持多选标签，实现灵活的任务分类（如 #Work, #Urgent）。

#### 2. 高级筛选与排序 (Advanced Filtering)
- [x] **全能筛选栏**: 
    - **搜索**: 支持按任务标题模糊搜索。
    - **标签筛选**: 支持按多个标签组合筛选任务。
- [x] **智能排序**: 
    - 支持按 **创建时间**、**截止时间**、**优先级**、**状态** 进行排序。
    - 一键切换 **升序/降序**。
- [x] **全局生效**: 筛选逻辑同时作用于 List View 和 Kanban View，体验统一。

#### 3. 数据导出 (Data Export)
- [x] **自主权**: 
    - 用户可在 Profile 页面导出 **任务列表** (`my_tasks.csv`) 和 **专注记录** (`my_focus_records.csv`)。
- [x] **标准格式**: 
    - 后端使用 Apache Commons CSV 生成标准格式文件，兼容 Excel/Numbers。

#### 4. 子任务系统 (Subtasks)
- [x] **无限拆解**: 
    - List View 支持展开任务，查看和管理子任务。
    - 支持快速添加、勾选完成、删除子任务。
- [x] **可视化进度**: 
    - Kanban 卡片上动态展示子任务完成进度条（如 `2/5`），直观把控任务状态。

#### 5. 高阶统计与分析 (Focus Analytics Pro)
- [x] **Heatmap (热力图)**: 
    - 类似 GitHub 的年度专注热力图，直观展示每日专注强度。
    - 支持深/浅色模式自动适配，针对低时长记录优化了色阶显示。
- [x] **精准时长**: 
    - 重构了时长统计逻辑，从简单的“次数×25”升级为后端精确聚合秒数，数据更真实。
- [x] **标签分析 (Tag Analysis)**: 
    - 新增饼图统计，直观展示用户在不同标签（如 #Coding, #Reading）上的时间分布。
    - 补全了“高阶统计”模块的最后一块拼图。

#### 6. 沉浸式体验与交互 (Immersive & UX)
- [x] **Zen Note (防走神便签)**: 
    - 全屏专注模式下新增快速记录板，捕捉灵感而不中断心流。
- [x] **拖拽排序 (Drag & Drop)**: 
    - 支持在 List View、Kanban View 和 Dashboard 中通过拖拽自由调整任务顺序。
    - 智能处理手动排序与自动筛选的冲突。
- [x] **点击即编辑**: 
    - 任务列表和卡片支持点击直接编辑，操作更流畅。

#### 7. 系统级优化 (System)
- [x] **深色模式 (Dark Mode)**: 
    - 全局适配 Element Plus 深色主题，包括图表配色与自定义组件。
- [x] **列表视图增强 (List View Pro)**:
    - **分页系统**: 新增客户端分页功能，提升长列表性能与体验。
    - **智能排序**: 已完成任务自动沉底并按时间排序；拖拽排序增加防误触机制（禁止拖入已完成区域）。
- [x] **看板视图增强 (Kanban View Pro)**:
    - **列内滚动**: 限制列高度并支持内部滚动，避免页面无限拉长。
    - **混合排序**: 默认按截止日期排序，同时支持手动拖拽微调顺序。

## 🛠 技术架构 (Technical Architecture)

- **前端**: Vue 3 + Vite + Element Plus + Pinia + Axios + ECharts + Sass + VueDraggable + html2canvas
- **后端**: Spring Boot 3.2.3 + MyBatis 3.0.3 + MySQL 8.0 + Apache Commons CSV
- **数据库**: 新增 `tags`, `task_tags`, `subtasks` 表，共 9 张核心表。
